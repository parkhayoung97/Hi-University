# Hi-University
- 프로젝트 소개: 대학의 학사관리 시스템과 인터넷 커뮤니티 시스템을 결합한 웹 사이트
- 개발 인원: 5명
- 담당 역할: 메인화면 및 로그인 CSS,성적 리스트 업데이트 페이지 및 학생 성적 불러오기
- 개발 기간: 2023.04.10 ~ 04.21
